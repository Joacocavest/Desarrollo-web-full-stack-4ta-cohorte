Ejercicio práctico Nº 13

Antes que nada:

Para desarrollar el siguiente ejercicio debe haber desarrollado previamente el ejercicio anterior: Trabajo práctico Nº 12

Consigna

  1. Copie la carpeta del ejercicio práctico anterior desarrollado en su computadora (ejercicio_practico12), y péguela con el nombre de ejercicio_practico13
  2. Abra el archivo index-tp13.html que se incluye en la carpeta de esta consigna y copie todo el código que ahí se encuentra.
  3. Utilice Visual Studio Code para trabajar con la carpeta copiada ejercicio_practico13 y abra el archivo index.html de dicha carpeta.
  4. Pegue el código copiado anteriormente en el área main del documento, a continuación de lo que ya había desarrollado.
  5. Proceda igualmente como en el ejercicio anterior (TP12), creando los estilos solicitados y combinando con los estilos ya creados.
  6. Para este trabajo práctico, deberá entregar la carpeta del ejercicio en formato .zip con el nombre tp13_nombre_suyo
  7. Fecha límite de entrega: martes 25/03 14:59hs.
