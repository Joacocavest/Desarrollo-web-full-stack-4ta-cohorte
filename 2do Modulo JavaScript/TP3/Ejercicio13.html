Ejercicio Nro. 13:

Realizar un programa que permita ingresar el importe total de una compra que realiza un cliente (esto lo
puede hacer con un input numérico) y sobre el mismo aplicarle un descuento del 15%. Mostrar (en el
documento HTML) el importe total de la compra y el descuento del 15% como así también el importe restante
final.

Nota: puede construir una arrow function que reciba como parámetro el importe total de la compra y que retorne el importe del descuento.
