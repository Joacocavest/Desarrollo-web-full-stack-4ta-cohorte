Ejercicio 16

Realizar un programa que permita ingresar el importe total de una compra y aplicar descuentos según
los totales de compra.
Si el monto es menor a $10.000, no tiene descuento.
Si el monto está entre $10.000 y $50.000, se aplica un 10% de descuento.
Si el monto es mayor a $50.000, se aplica un 20% de descuento.

Nota: puede construir una arrow function que reciba como parámetro el importe total y que retorne el
descuento.
Mostrar el importe final a pagar luego del descuento.
